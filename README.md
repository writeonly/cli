# Aliases

## Install aliases

```bash
curl -s https://raw.githubusercontent.com/writeonly/cli/master/bash_aliases_install.sh | bash
```

## Update aliases
```bash
curl -s https://raw.githubusercontent.com/writeonly/cli/master/bash_aliases_update.sh | bash
```
or
```bash
update_aliases
```
